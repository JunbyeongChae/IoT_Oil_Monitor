# ⛽ 스마트 오일 탱크 잔량 모니터링 시스템 (IoT Oil Tank Level Monitor)

## 📌 프로젝트 요약 (Summary)

### 문제 정의

기존 단독주택 기름보일러의 잔량 확인 방식(수동, 외부에 위치)은 번거롭고, 겨울철 연료 부족 시점에 예측이 어려워 불편함이 컸습니다. 특히, 본 프로젝트의 대상 탱크는 깊이 **1200mm**로 일반적인 센서 적용에 한계가 있었습니다.

### 해결책 및 핵심 목표

ESP32 기반의 IoT 시스템을 구축하여 **실시간 원격 모니터링**을 구현하고, **실외 환경에 최적화된 방수 설계(풀옵션 패키징)**를 통해 잔량 확인의 불편함을 근본적으로 해소합니다.

---

## ✨ 주요 기능 (Features)

* **실시간 잔량 모니터링:** Blynk 앱을 통해 잔량(%) 및 빈 공간(cm)을 언제 어디서든 실시간으로 확인 가능.
* **잔량 부족 알림:** 잔량이 **15% 이하**로 떨어질 시 모바일 푸시 알림(Notification) 자동 발송.
* **비접촉식 고정밀 측정:** AJ-SR04M 방수 초음파 센서를 활용하여 깊이 120cm의 탱크 잔량을 정확하게 측정.
* **실외 환경 내구성:** IP66 방수 정션 박스 및 케이블 글랜드를 적용한 풀옵션 패키징 설계.

---

## 🛠️ 기술 스택 (Tech Stack)

| 구분 | 구성 요소 | 상세 스펙 |
| :--- | :--- | :--- |
| **마이크로컨트롤러** | ESP32-WROOM-32 | C-Type, 38핀, Wi-Fi 802.11 b/g/n |
| **센서** | AJ-SR04M | 방수형 초음파 모듈 (측정 거리 2cm ~ 450cm) |
| **개발 언어** | C++ | Arduino Sketch |
| **IoT 플랫폼** | Blynk IoT | 데이터 시각화, 알림 기능 (Virtual Pin V0, V1 사용) |
| **전원** | USB-C Power | 5V 3A 어댑터 및 20AWG C-Type 케이블 |
| **하드웨어 설계** | IP66, 38핀 스크류 터미널 쉴드 | 실외 환경 최적화 및 견고한 결선 |

## 🚀 구현 및 설치 과정 (Assets Preview)

****

1.  **초기 기능 구현:** 브레드보드를 이용해 ESP32와 AJ-SR04M 센서를 연결하고, Blynk 통신 및 잔량 계산 알고리즘을 구현하여 기능 검증 (로드맵 Phase 2).
